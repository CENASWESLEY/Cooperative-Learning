เทคโนโลยี

Technology

หลักสูตรเทคโนโลยีนี้จำทำให้คุญสามารถนำความรู้ไปประยุกต์ใช้ได้ในแต่ละสายงานอาชีพ เพราะว่าหลักสูตรของเรามีคอร์สให้เลือกเรียนมากมายโดยความรู้ในแต่ละบทนั้นถูกคัดสรร จากผู้เชี่ยวชาญโดยเฉพาะ ซึ่งสามารถเรียนได้ทั้งระดับผู้ที่พึ่งเริ่มต้นศึกษาไปจนถึงผู้ที่มี ความรู้ระดับกลางที่พร้อมจะต่อยอดให้ไปถึงระดับที่สามารถนำความรู้มาประยุกต์ใช้เองได้ และสร้างสรรค์สิ่งใหม่ๆจากองค์ความรู้เดิม

คอร์สเรียน

Web Development (Full Stack)

การพัฒนาเว็ปไซต์ทั้งด้าน Frontend และ Backend ด้วย HTML,CSS และ Javascript ซึ่งคอร์สนี้จะพาคุณลงมือทำเว็ปไซต์แบบครบวงจร

เนื้อหา : 10 บทเรียน

ผู้สอน : สมาคมสารสนเทศ

Internet of thing (IOT)

เรียนรู้การใช้ IoT เพื่อประยุกต์ใช้ในเชิงลึกของภาคอุตสาหกรรม เพื่อให้ เข้าใจหลักการทำงานของ IoT ร่วมกับอุปกรณ์อื่น

เนื้อหา : 20 บทเรียน

ผู้สอน : สมาคมสารสนเทศ
